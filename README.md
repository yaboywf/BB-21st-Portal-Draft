## What is it ?
This website helps to facilitate administrative tasks such as creating 32A results, user management and awards tracking.

## Installation: 
To run, please install the following:
- python
- flask
- flask_compress
- flask_caching
- flask_socketio
- sqlite3
- bcrypt
- uuid
- os
- functools
- typing
- datetime
- flask_login
- cryptography.fernet
